## 🏗️ 架构设计

### 当前架构
```
输入 (logλ, p) → ParametersEmbedding → Transformer 层 → PredictionHead → 输出 (I, continuum)
```

### 改进架构 (PhysFormer 风格)
```
输入 (logλ, p) → ParametersEmbedding → Transformer 层 
                                          ↓
                              PhysicalQuantitiesHead
                                          ↓
                              (κ, σ, T) 物理量
                                          ↓
                              RTE 数值积分
                                          ↓
                              角度积分
                                          ↓
                              输出 (I, continuum)
```

## 🔧 技术要点

### JAX 可微分实现
- 所有物理公式使用 `jax.numpy`
- 使用 `jax.grad` 验证梯度
- 避免 Python 控制流，使用 `jax.lax.cond`

### 数值稳定性
- 吸收系数使用 `softplus` 确保非负
- 光学深度使用累积和
- 指数函数使用裁剪

### 性能优化
- 使用 `jax.vmap` 批量处理波长
- 使用 `jax.jit` 编译关键函数
