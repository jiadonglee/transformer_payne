## 🏗️ 技术架构

### 核心组件

```
┌─────────────────────────────────────────────────────────┐
│              物理嵌入 TransformerPayne                    │
├─────────────────────────────────────────────────────────┤
│  输入层                                                  │
│  (logλ, p) - 波长 + 恒星参数                             │
├─────────────────────────────────────────────────────────┤
│  ParametersEmbedding                                     │
│  95 维 → 256 维 × 16 tokens                               │
├─────────────────────────────────────────────────────────┤
│  Transformer Layers (8 层)                                │
│  MHA + FeedForward + RMSNorm                            │
├─────────────────────────────────────────────────────────┤
│  PhysicalQuantitiesHead                                  │
│  ↓ 吸收系数 κ (波长相关)                                  │
│  ↓ 散射系数 σ (波长相关)                                  │
│  ↓ 温度 T (深度相关)                                      │
├─────────────────────────────────────────────────────────┤
│  OpticalDepthComputer                                    │
│  τ = cumsum(κ + σ)                                      │
├─────────────────────────────────────────────────────────┤
│  RadiativeTransferSolver                                 │
│  dI/dτ = I - S                                          │
│  S = Planck(T)                                          │
├─────────────────────────────────────────────────────────┤
│  AngularIntegration                                      │
│  F = 2π ∫ I μ dμ                                        │
├─────────────────────────────────────────────────────────┤
│  输出层                                                  │
│  (I_λ, F_λ) - 强度 + 通量                                │
└─────────────────────────────────────────────────────────┘
```

### 创新点

1. **物理过程内嵌** - 不是外部损失约束，而是生成机制的一部分
2. **可微分 RTE** - 整个辐射转移过程可微分，支持端到端训练
3. **低维潜在空间** - 学习物理可解释的中间表示 (256 维)
4. **兼容性设计** - 可切换原始/物理模式，可加载现有权重

## 🚀 使用示例

### 快速开始

```python
from transformer_payne_physics import create_physics_embedded_model
from flax.core.frozen_dict import freeze
import jax.numpy as jnp

# 创建模型
model, params = create_physics_embedded_model(use_physics=True)

# 准备输入
log_wavelengths = jnp.log10(jnp.linspace(4670, 4960, 100))
solar_params = jnp.array([5777, 4.44, 0.0, 0.0])

# 前向传播
variables = {"params": freeze(params)}
spectrum, physics = model.apply(
    variables,
    (log_wavelengths, solar_params),
    train=False,
    return_physics_outputs=True,
)

print(f"光谱形状：{spectrum.shape}")
print(f"吸收系数：κ ~ {physics['kappa'].mean():.2e} cm⁻¹")
print(f"温度：T ~ {physics['T'].mean():.1f} K")
```

### 训练

```bash
python train_physics_embedded.py \
    --data_dir /path/to/phoenix \
    --output_dir ./output \
    --epochs 200 \
    --batch_size 32 \
    --physics_weight 0.1
```
